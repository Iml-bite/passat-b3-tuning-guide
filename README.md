# 🚗 Volkswagen Passat B3 - Руководство по ремонту

Интерактивный веб-сайт с руководством по ремонту и обслуживанию Volkswagen Passat B3 (1988-1993).

## 🎯 Возможности

### 📖 Руководство по ремонту
- Подробные инструкции по ремонту двигателя
- Руководство по обслуживанию трансмиссии
- Инструкции по ремонту подвески
- Руководство по тормозной системе
- Инструкции по электрике

### 🔍 Интерактивная диагностика
- Выбор категории проблемы (двигатель, трансмиссия, подвеска, тормоза, электрика)
- Описание симптомов
- Автоматический анализ вероятных причин
- Рекомендации по устранению
- Оценка стоимости ремонта

### 💰 Калькулятор стоимости ремонта
- Выбор необходимых работ из каталога
- Автоматический расчёт стоимости
- Расчёт времени выполнения работ
- Печать сметы на ремонт
- Автоматическое сохранение выбора в браузере

## 🚀 Запуск

Просто откройте `index.html` в браузере. Сайт не требует сервера и работает полностью на клиентской стороне.

```bash
# Или запустите локальный сервер
python3 -m http.server 8000
# Откройте http://localhost:8000
```

## 🔧 Разработка

Для добавления новых работ в калькулятор отредактируйте объект `repairPrices` в `calculator.js`:

```javascript
{
    id: "work-id",
    name: "Название работы",
    price: 5000,  // в рублях
    time: 2       // в часах
}
```

Для добавления новых симптомов в диагностику отредактируйте объект `diagnosticData` в `diagnostic.js`.

## 📄 Лицензия

## ⚠️ Дисклеймер

Указанные цены являются ориентировочными и могут варьироваться в зависимости от региона, сервиса и качества запчастей. Перед началом ремонта рекомендуется проконсультироваться со специалистами.
